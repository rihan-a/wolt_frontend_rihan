## Run the App Locally

Go to the project directory

```bash
  cd wolt_frontend_rihan
```

Install project dependencies

```bash
  npm install
```

Start the web app

```bash
  npm start
```

```
Open http://localhost:3000 to view it in the browser.
```

## Running Tests

To run tests, run the following command

```bash
  npm test
```
